Command-Based Git Workflow

git clone
Copy a remote repo to your computer.
git add
Stage changes for commit.
git commit
Save staged changes locally.
git push
Upload commits to remote repo.
git pull
Fetch and merge remote changes.
git merge
Combine branches together.
git checkout
Switch branches or restore files.
